   MÉTHODE 3 : LIGNE DE COMMANDE (Avec Token)
═══════════════════════════════════════════════════════

Si vous avez un token Cloudflare valide :

1. Créer token sur :
   https://dash.cloudflare.com/profile/api-tokens
   → "Create Token"
   → Template "Edit Cloudflare Workers"
   → Account → Cloudflare Pages → Edit

2. Dans terminal :
   export CLOUDFLARE_API_TOKEN="votre-token-ici"
   cd /home/user/webapp
   npx wrangler pages deploy dist --project-name gxomoissyprocedures
